#+title: CowardCoin v4

* About the Project
This is the 4th edition of CowardCoin, my Discord bot that creates coins at random intervals :)

** Commands
- =coin count= - see how many coins you currently have
- =coin give= - give coins to other people
- =coin leaderboard= - see where you rank
- =coin get= - grab a coin that's sitting there waiting to be collected

* Implementation Info 
** Project Structure
Rust's memory safety means that the Discord bot itself will be unable to access the data - and I'm getting around that by running it on two separate threads.
When the bot receives a command, it will send an enum of the request to the backend, which will reply with an SRC_rust{Option<String>} object - the message to reply to the command with.
